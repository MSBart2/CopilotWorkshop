## Key Visual Elements

### Color Coding
- **Red/âŒ** - Problems, before state (`bg-red-50 dark:bg-red-900`)
- **Green/âœ¨** - Solutions, after state (`bg-green-50 dark:bg-green-900`)
- **Blue** - Concepts, information (`bg-blue-50 dark:bg-blue-900`)
- **Purple** - Actions, tasks (`bg-purple-50 dark:bg-purple-900`)
- **Yellow** - Transitions (`bg-yellow-50 dark:bg-yellow-900`)

### Layout Patterns
- **Cover** - Title slides with large text and emojis
- **Two-cols** - Side-by-side comparisons
- **Center** - Quotes, key messages, philosophy
- **Default** - Standard content with headings and lists
- **Grid** - Multiple cards or boxes (2-4 columns)

### Typography
- **Headings:** Large, bold, clear hierarchy
- **Body:** Clean, readable, adequate spacing
- **Quotes:** Italic, centered, with attribution
- **Code:** Monospace with syntax highlighting

### Spacing
- Generous whitespace for readability
- Consistent padding in cards (p-4, p-6)
- Gap between elements (gap-4, gap-6, gap-8)

## How It Renders

When you run `npm run dev`, Slidev creates:

1. **Interactive web presentation** - Navigate with arrow keys
2. **Presenter mode** (press P) - Speaker notes, timer, next slide
3. **Drawing mode** (press D) - Annotate slides live
4. **Remote control** - QR code to control from phone
5. **Export options** - PDF, PPTX, PNG, static site
