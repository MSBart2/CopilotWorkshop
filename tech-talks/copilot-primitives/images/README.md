# Images Directory

This directory contains visual assets for the Copilot Primitives tech talk.

## Required Diagrams

1. **architecture-primitives-pyramid.svg** - Shows the four primitives in a pyramid structure with Instructions at the base (always on), Skills in the middle (on-demand), Custom Prompts above (user-invoked), and Agents at the top (orchestration)

2. **custom-instructions-reference.png** - VS Code Copilot Chat showing References dropdown with `.github/copilot-instructions.md` highlighted

3. **primitives-flow-diagram.svg** - Data flow showing how user requests go through context assembly (Instructions → Skills → Prompts → Agents) to the language model

4. **primitives-quick-reference.svg** - Table showing each primitive type with columns: Name, Scope, File Path, Use Case
